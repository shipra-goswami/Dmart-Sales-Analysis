#  DMart Sales Dashboard 2024 — Excel Project

An interactive Excel dashboard built on Superstore retail sales data — designed with a DMart-inspired theme to replicate real-world Business Intelligence workflows.

---

## Tools & Skills
**Microsoft Excel** — Pivot Tables · Slicers · Charts · Timeline Filter · Data Cleaning · Dashboard Design

---

##  Dataset
- **Source:** Sample Superstore Sales Dataset
- **Records:** 9,995 transactions
- **Fields:** Order ID, Category, Sub-Category, Region, Segment, Ship Mode, Product Name 

---

##  What I Built

| Feature | Details |
|--------|---------|
| Pivot Tables | 5 — by Category,sub-category , Region, Segment, Ship Mode |
| Charts | 5 — 4 Pie Charts + 1 Bar Chart |
| Slicers | State filter + Sub-Category filter |
| Timeline | Filter by Ship Date (monthly/yearly) |

##  Key Findings

| Metric | Value |

| Total Revenue | ₹22,97,201 |

| Top Region | West — ₹7,25,458 |
| Lowest Region | South — ₹3,91,722 |
| Top Segment | Consumer — ₹11,61,401 (50.5%) |
| Top Sub-Category | Phones — ₹3,30,007 |
| Most Used Ship Mode | Standard Class — ₹13,58,216 |

---

## Business Insights
- **Technology** generates more revenue than Furniture and Office Supplies combined
- **South region underperforms** vs West by ~46% — opportunity for targeted marketing
- **Consumer segment** drives 50.5% of total revenue — retention strategy is critical
- **Standard Class shipping** is most preferred — indicates price-sensitive buyer behaviour
